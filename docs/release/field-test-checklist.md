# BreedLog Field-Test Checklist (Web/PWA)

Use this checklist during controlled testing.

## Core setup
- [ ] Install/open BreedLog PWA on phone.
- [ ] Open BreedLog on desktop browser.
- [ ] Use the same access code on one phone and one desktop.
- [ ] Log out and back in on both devices.

## Data and offline flow
- [ ] Add animal while online.
- [ ] Add animal while offline.
- [ ] Switch airplane mode on/off.
- [ ] Confirm no duplicate animals appear after reconnect.
- [ ] Edit animal and add weaning data.
- [ ] Confirm KW-style tag prefix behavior works as expected.

## Health and analysis
- [ ] Add health event manually.
- [ ] Open Health Plan.
- [ ] Use a Health Plan action button to record an event.
- [ ] Open Analysis tab and verify reports load.

## Import/export
- [ ] Export CSV.
- [ ] Download/import CSV template.
- [ ] Export PDF where available.

## Large herd and performance
- [ ] Browse large animal list and use **Load More**.
- [ ] Use search/filter and verify results remain correct.

## App persistence/sync verification
- [ ] Reload/refresh app.
- [ ] Confirm data remains after refresh.
- [ ] Go offline/online again and confirm sync catches up.

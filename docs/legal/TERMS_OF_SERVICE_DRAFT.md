# BreedLog Terms Of Service Draft

Version: terms-of-service-draft-2026-07-13
Status: implementation draft requiring professional legal review before production use.

## Service

BreedLog provides livestock record keeping, performance reporting, exports, backups and optional AI-assisted analysis for farm management.

## Plans

BreedLog has exactly two plans: Free and Premium. There is no trial plan. Backend entitlement state is authoritative; client-side feature hiding is not the source of truth.

## User Responsibilities

Users are responsible for lawful use, accurate data entry, safe backup storage, account security and independent verification of management decisions. BreedLog must not be used as a substitute for veterinary, legal, tax or financial advice.

## Platform Limitations

Production activation of payments, Android distribution, Windows signing and domain deployment requires external credentials and approvals. Those dependencies must be completed before public launch.

## Review

This document must be reviewed by qualified legal counsel before production launch.

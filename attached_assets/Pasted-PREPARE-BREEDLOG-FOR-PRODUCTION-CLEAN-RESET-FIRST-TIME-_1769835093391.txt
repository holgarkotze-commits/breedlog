B) FIRST-TIME USER ONBOARDING FLOW
====================================================

Goal:
When a brand-new user opens BreedLog for the first time (no animals yet),
show a short guided onboarding that helps them set up fast.

B1) Detect first-time user
Create a reliable check:
- “first run” if:

Create a 4-step onboarding wizard (mobile friendly):

Step 2: Farm Setup

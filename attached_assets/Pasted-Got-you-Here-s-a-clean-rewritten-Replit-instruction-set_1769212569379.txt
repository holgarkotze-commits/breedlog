## Replit Instructions: Update “New Lamb Entry” & PDF Export (Per Animal)

### 1) Update the **New Lamb Entry** tab – required data fields

Extend the **New Lamb Entry** form to include the following fields. These fields must be saved per lamb and remain editable later.

#### A) Core Identification & Birth Data

* **Lamb ID**

  * Type: Text / Alphanumeric
* **Date of Birth**

  * Type: Date picker
* **Birth Status**

  * Type: Dropdown
  * Options:

    * Single
    * Twin
* **Sex**

  * Type: Dropdown
  * Options:

    * Ram
    * Ewe
* **Sire (Father)**

  * Type: Searchable dropdown or text (linked to ram records if available)
* **Dam (Mother)**

  * Type: Searchable dropdown or text (linked to ewe records if available)

---

#### B) Growth & Weaning Data

* **100-Day Weigh Date**

  * Type: Date picker
* **100-Day Weight (kg)**

  * Type: Number input (decimals allowed)
* **Weaning Status – Multiple Lamb Loss**

  * Type: Dropdown
  * Purpose: indicate if a twin lamb’s sibling died before weaning
  * Options:

    * Normal
    * Sibling died before weaning

### 3) Validation rules

* **Lamb ID**, **Date of Birth**, **Sex**, **Sire**, and **Dam** are mandatory.

* Weight values must be numeric and greater than 0.
* **Weaning Status – Sibling died before weaning** should only be selectable if **Birth Status = Twin**.

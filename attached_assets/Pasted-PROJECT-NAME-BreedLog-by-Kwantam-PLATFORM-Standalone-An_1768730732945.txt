1.3 Grouping rules (for comparisons and future EBV logic)

Lambs can only be compared if born within a 2-month window.

Therefore mating period should never be longer than 2 months.

Use a Lambing Season Code, example: “07H” for 2007 Autumn, and if longer create “07H2”.

Build a validator: warn user if mating window > 2 months, and suggest creating a second lambing season code group.

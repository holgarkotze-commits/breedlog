1. User clicks Add Animal.
2. User enters animal details.
3. User uploads/selects an image.

5. The full uploaded image is visible.
6. A visible rectangular crop box appears over the image.
7. User can resize the crop box by dragging:
   - corners

   - top/bottom sides
8. User can click/hold inside the crop box/image area and drag to reposition the crop area over the animal.
9. User clicks Save Crop.
10. The cropped image is saved into the animal image/photo field.
11. The animal card/profile displays the saved cropped image.

When editing an existing animal profile:

1. User opens animal profile.
2. User changes/uploads a new image.
3. Same crop dialog opens.
4. User crops image.
5. User clicks Save Crop.
6. The existing animal profile image updates only after saving.

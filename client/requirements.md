## Packages
recharts | For weight tracking and performance charts
framer-motion | For smooth page transitions and rugged interaction animations
date-fns | For date formatting in records
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- Tailwind config needs to extend colors for the "Black & Yellow" theme.
- Fonts: 'Chivo' for headings (Industrial/Rugged), 'Inter' for UI.
- Auth is handled via Replit Auth, using `useAuth` hook.
- AI Valuation requires authenticated user.

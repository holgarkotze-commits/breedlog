## Notes
- Tailwind config needs to extend colors for the "Black & Yellow" theme (#0B0B0B background, #FFC300 accent).
- Fonts: 'Chivo' for headings (Industrial/Rugged), 'Inter' for UI.
- Auth is handled via Replit Auth, using `useAuth` hook.
- AI Valuation requires authenticated user.
- Offline-first UX principles applied (optimistic UI where possible).
